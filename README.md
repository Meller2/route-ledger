# Route Ledger

**Route Ledger** — премиальное Android-приложение для курьера: смены, доход, расходы, районы и личная аналитика без табличного ада.

Дизайн-направление: **Quiet Premium** — молочный фон, графит, тёплое золото, sage-акценты, большие карточки и спокойная финансовая эстетика. Никакого неона, киберпанка и визуального шума.

## Что уже есть в MVP

- Kotlin + Jetpack Compose
- Material 3
- Hilt
- Room
- DataStore подключён как зависимость для будущих настроек
- Русский интерфейс
- Нижняя навигация:
  - **Пульс** — главный финансовый дашборд
  - **Смена** — текущая/последняя смена и заказы
  - **Доход** — история смен и график
  - **Районы** — аналитика районов
  - **Ещё** — настройки и будущие интеграции
- Демо-данные для красивого первого запуска
- Заготовка под карты через `MapProvider`, пока без Google Maps/Yandex Maps API key
- GitHub Actions для сборки debug APK

## Почему карты не подключены сразу

Карты специально вынесены за интерфейс `MapProvider`. Сейчас используется `NoOpMapProvider`, чтобы приложение не требовало API key и не ломалось на первом запуске. Позже можно добавить отдельную реализацию под Google Maps или Яндекс Карты и подключить её через DI.
