# Как затестить Route Ledger без компа

1. Открой репозиторий: https://github.com/Meller2/route-ledger
2. Перейди во вкладку **Actions**.
3. Открой последний запуск **Android CI**.
4. Дождись зелёной галочки.
5. Внизу страницы запуска найди **Artifacts**.
6. Скачай `route-ledger-debug-apk`.
7. Распакуй zip на телефоне.
8. Установи `app-debug.apk`.

Важно: Android может попросить разрешить установку APK из браузера/файлового менеджера. Это нормально для debug-сборки.

Если сборка упала, смотри лог шага **Build debug APK** — там будет причина.
